CSS

Once you have statred dabbilng in Html, you will probably be more intrested in adding more power to your web . Css is the best way to do that and css is the language thats lets you apply changes to your entire page without having use lots of inline Html styles. We will go over to create an inline stlyesheet so that you can pratice your css skills, then we will move on to ten simple examples that will shwo you how to do a few basic things. From there, your imagination is the limit

The cool thing about styling with css is that you do not have to specify a style every time you create an document. You can just say "all paragraphs should have this particular styling" and you are good to go. Here is an exapmle of how one may do that. let's say you waht every paragraph (that's everything with a Html tag) on your page to be slightly larger than all. And dark grey, instead of blacy. Here is how you would do that with CSS: p{ font-size: 120%; color: dimgray; }

Okay, so now that we have seen how to make a change to every paragraphs, let us see how we can be a bit more active. Let us create a designation for paragraphs that should be in small caps. Here is how we can do that: p.smallcaps{ font-variant: small-caps; }
